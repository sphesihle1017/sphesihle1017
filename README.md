## Azure Hotels — Booking & Management System

**Tech:** ASP.NET MVC, C#, Entity Framework, SQL Server, HTML, CSS, JavaScript, Bootstrap

A full-featured hotel management web application with role-based access control, providing seamless experiences for both administrators and customers.

**Key Features:**
- **Role-Based Authentication:** Separate secure logins and dashboards for administrators and customers
- **Admin Panel:** Full CRUD operations for rooms, comprehensive booking management, and customer overview
- **Customer Portal:** Room browsing, booking creation, booking history, and cancellation capabilities
- **Database Integration:** Entity Framework (Code-First) approach modeling relationships between rooms, bookings, and users in SQL Server
- **Responsive UI:** Bootstrap-powered interface optimized for all devices
